# 存放用户画像模版

| 编号 | 场景 |
| ---- | ---- |
| 1 | personalInfo |
| 2 | travel |
| 3 | entertainment |
| 4 | shoppingPreferences |
| 5 | edu |
| 6 | health |

## 命名规则：
* 文件名格式：xy_ab.json
  * xy：场景编号
    * x：场景1编号
    * y：场景2编号
  * ab：场景名称
    * a：场景x的第a个模版
    * b：场景y的第b个模版
* 例：
  * 12_11.json代表编号为1(personalInfo)、2(travel)的混合场景各自的第一个`元模版`
